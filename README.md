# 729 Grams Coffee

A standalone React + TanStack Start project for 729 Grams Coffee roastery & café.

## Development

Make sure you have Node.js installed.

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Tech Stack

- React 19
- TanStack Start & TanStack Router
- TypeScript
- Vite & Tailwind CSS
